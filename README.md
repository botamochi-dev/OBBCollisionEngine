# OBBCollisionEngine

## 概要

本リポジトリは、3次元空間における**OBB (Oriented Bounding Box) 同士の衝突判定&衝突応答**に特化した軽量なライブラリ/エンジンです。

## 動作確認済みバージョン
- 1.19.4

## 機能

### 衝突応答

高速かつ正確な衝突検出のため、物理演算で広く用いられる**分離軸定理 (Separating Axis Theorem, SAT)** に基づいて実装されています。
現時点では**「衝突しているか否か (bool)」**の判定のみをサポートしており、衝突応答（Response）や詳細な衝突情報（接触点、貫通深度など）の計算は今後の課題としています。
